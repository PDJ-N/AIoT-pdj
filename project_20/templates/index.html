Raspberry Pi LED Dashboard

LED 원격 제어

라즈베리파이 GPIO에 연결된 LED를 브라우저에서 간단하게 켜고 끌 수 있는 제어 화면입니다. 무거운 효과는 줄이고, 버튼은 크게 배치해서 작은 화면에서도 보기 쉽도록 구성했습니다.

아래 버튼을 누르면 `name="led"` 값이 서버로 전달됩니다. `main20-1.py` 에서 이 값을 받아 LED ON/OFF 로직과 연결하면 됩니다.

터치 화면이나 모바일 브라우저에서도 누르기 쉽도록 버튼 크기를 크게 잡았습니다.

폼 전송 경로는 `/data`로 유지해 두었습니다. `main20-1.py` 라우트에 맞게 필요하면 `action`만 바꾸면 됩니다.
